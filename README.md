# VoxPrompter

> 语音提词器 / VAD 节奏跟随 + KTV 歌词展示 / 单文件 HTML 开箱即用

## 使用

打开 `index.html` → 粘贴文案 → 进入提词模式 → 开口朗读

## 核心逻辑

- **朗读**：说一句话 → 安静 1 秒 → 自动前进到下一句
- **回退**：安静后说一个孤立短词（如"哈"）→ 回退一句
- **录音**：点击 ● 按钮同步录制口播音频

## 参数可调

灵敏度 / 静音判定超时 / 短语阈值 / 回退窗口 — 右上角 ⚙ 面板实时调节

## 快捷键

| 按键 | 功能 |
|------|------|
| ↑ / ← | 回退 |
| ↓ / → | 前进 |
| Space | 暂停/恢复麦克风 |
| Esc | 退出提词 |

## 技术

纯 Web Audio API（VAD），零语音识别，零外部依赖。推荐 Chrome/Edge。
